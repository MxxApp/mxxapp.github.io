## 串流分享多人使用，类似云电脑方案

### 软件Duo  
- 免费支持一个实例，显示输出限制30Hz  
- 官网：[https://github.com/DuoStream/Duo](https://github.com/DuoStream/Duo)
- 核心：Duo 支持多用户远程串流登录，与 Moonlight 搭配，实现云桌面功能。
### 搭建教程：
1. **Win增加一个新用户，可空密码，Duo新版已支持空密码**  
2. **Win不支持空密码登陆，需要修改组策略**  
    - 运行`gpedit.msc`打开组策略编辑器  
    - 路径：`计算机配置 > Windows 设置 > 安全设置 > 本地策略 > 安全选项`  
    - 右边拉倒底部，找到 `账户：使用空密码的本地账户只允许进行控制台登陆`，修改成禁用  
3. **切换到新创建的用户，进去配置好桌面，现有Duo版本建议配合Sandboxie使用**  
4. **防止其它用户关机重启主机，切回主用户，运行mmc打开控制台根节点**
    - 左上角文件 > 添加/删除管理单元 > 组策略对象编辑器
    - 添加 > 浏览 > 用户 > 选择新添加的用户 > 配置新用户单独的组策略
    - 路径：用户配置 > 管理模版 > "开始"菜单和任务栏
    - 找到 `删除并阻止访问"关机"、“重新启动”、"睡眠"和"休眠"命令`，修改成启用
    - 可选：找到 `删除"开始"菜单上的"注销"`，修改成启动
    - 文件 > 保存 即可  
5. **安装Duo,运行Duo Manager>+New>按需设置，完成后按Start启动**  
    - 等待右侧Pair New Client亮起，点击Open Sunshine配置账号密码后，即可用moonlight连接了。  
6. **和主用户共用程序，配合Sandboxie使用更好，等待Duo更新沙箱功能。**  
    - Sandboxie具体使用方法自行搜索，以下是一些使用记录

## Sandboxie快捷方式使用记录

- 普通模式 示例：
    ```
    目标："C:\Program Files\Sandboxie-Plus\Start.exe" /box:DefaultBox D:\Game\test.exe
    起始位置：D:\Game\
    ```
- 最小化启动 -silent 示例：
    ```
    目标："C:\Program Files\Sandboxie-Plus\Start.exe" /box:DefaultBox D:\Game\Steam\steam.exe -silent
    起始位置：D:\Game\Steam\
    ```
- 管理员启动 /elevate 示例：
    ```
    目标："C:\Program Files\Sandboxie-Plus\Start.exe" /box:DefaultBox /elevate "D:\Game\HMCL-3.12.0.328.exe"
    起始位置：D:\Game\
    ```

## 断开串流连接，方便用户退出或远程关闭

### 创建vbs脚本，对于不懂如何断流的用户，可制作 Sunshine 控制API的快捷脚本：
```vbscript
' 1. 配置信息
Dim user, pass, url, auth, curlCmd
user = "用户名"
pass = "密码"
url = "https://127.0.0.1:31609/api/apps/close"

' 2. 构建认证和命令
' 注意：-s 是静默模式，-k 是忽略证书，-u 是账号密码，-H 是指定格式
auth = user & ":" & pass
curlCmd = "curl.exe -s -k -u " & auth & " -H ""Content-Type: application/json"" -X POST " & url

' 3. 执行命令而不显示窗口 (0 代表隐藏窗口)
Set shell = CreateObject("WScript.Shell")
shell.Run curlCmd, 0, True

' 可选：如果你想给用户一个简单的确认弹窗，可以去掉下面一行的注释
' MsgBox "远程连接已断开", 64, "提示"
```
- 保存后，创建快捷方式放到需要的位置即可。
